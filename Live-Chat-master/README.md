# Live-Chat

Live Chat is a real-time, live-chat software tool that helps e-commerce companies create a new sales channel by offering a superior online support. What better way to offer them help than through a quick chat on your website? Itâ€™s way faster than email and more efficient than phone. This project was developed using the programming language PHP, with additional functionality developed in jQuery, AJAX, HTML and CSS. 


To configure the project
1) Place the project in htdocs or www folder of XAMPP or WAMP respectively.
2) create db live_chat in phpMyAdmin.
3) import live_chat.sql to db.
4) Start the apache and mysql server to run.
